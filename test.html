How to Test the Extension:

  1. Make sure the Carbon Filter extension is installed and enabled in Chrome
  2. Click the extension icon in the toolbar to open the popup
  3. Make sure "Privacy Protection" is turned ON
  4. Check/uncheck different field types to test individual masking
  5. Add custom text in the "Custom Text to Hide" section
  6. Refresh this page to see the masking effects
  7. Hover over masked text to see the original content
